# Go + DevOps Learning Journey

A comprehensive, hands-on approach to learning Go programming for **DevOps engineers**.

Each module combines:
- Go language concepts
- Docker containerization
- Git/GitHub workflows  
- DevOps automation examples
- CI/CD integration

## Module Roadmap

| Module | Topic | Focus |
|--------|-------|-------|
| 0 | Environment Setup | Go, Docker, Git, GitHub CLI |
| 1 | Go Fundamentals | Variables, types, functions, error handling |
| 2 | Container Automation | Working with Docker via CLI and Go |
| 3 | Git Automation | Git operations with Go scripts |
| 4 | CLI Tools | Building DevOps tools with cobra |
| 5 | Concurrency | Goroutines, channels, distributed tasks |
| 6 | HTTP & APIs | Web services, REST, Docker APIs |
| 7 | Infrastructure | Container orchestration, K8s basics |
| 8 | Real Projects | Full DevOps automation tool in Go |

## Quick Start

\\\powershell
# 1. Navigate to workspace
cd ~\devops-go-journey\golang-devops-journey

# 2. Start with Module 0
gh issue list --milestone "Module 0"

# 3. Create your first Go program
go mod init github.com/rmansoura/devops-go

# 4. Track progress on GitHub
gh issue create --title "Completed: Module 1 Lesson 1" --label "completed"
\\\

## Learning Approach

Each module contains:
- **Theory**: Concepts and explanations
- **Examples**: Practical code samples
- **Challenges**: Real-world scenarios
- **Projects**: Build something useful
- **Tests**: Verify your understanding

## GitHub Tracking

All progress is tracked via:
- **Issues**: Learning tasks and challenges
- **Milestones**: Module completion
- **Project Board**: Kanban-style task management
- **Pull Requests**: Code review and learning
